# css 元素

## 分类

- 块状元素
- 行内元素

## 块状元素

- 属性
  - width
  - height
  - padding
  - margin
  - border
- 前后带有换行符，自占一行
- 一般作为容器、盒子使用
- 默认情况下，宽度与父元素同宽，高度根据内容而定，没有内容时 宽度为0
- 常见元素
  - h1 - h6
  - p
  - div
  - ul ol li

## 行内元素

- 不占独立的区域，仅仅靠自身的字体大小和图像尺寸来支撑结构
- 一般不可以设置高度、对齐等属性
- 控制页面中文本样式
- 常见元素
  - strong
  - b
  - em
  - i
  - del
  - s
  - ins
  - u
  - a
  - span

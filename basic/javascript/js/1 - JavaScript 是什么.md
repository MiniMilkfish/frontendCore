# 1 - JavaScript 是什么

## 是什么

Brendan Eich 花了10天设计完成的一门语言，大杂烩，借鉴并参考多个语言

 - 基本语法： C、Java
 - 数据结构： Java - 将值分为原始值和对象两大类
 - 函数的用法： Scheme、Awk 将函数作为第一等公民，并引入闭包
 - 原型继承模型： Self
 - 正则表达式： Perl
 - 字符串和数组处理： Python

它是一门动态、弱类型的编程语言，非常适合卖你想对象和函数式编程风格
    - 动态语言： 运行时确定数据类型
      - 静态语言： 编译时变量的数据类型就已经确定
    - 弱类型：变量类型由上下文确定
      - 强类型：变量的类型一旦确定就不能被转化
